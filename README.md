# 《绿色三角洲》半自动人物卡

用于《绿色三角洲TRPG》网团的Excel半自动人物卡。

最新版本：v0.4.5

亮点：
- 完整：展示DG人物卡的所有要点（属性、衍生属性、技能、适应性、特训、重要关系……）
- 简洁：一个屏幕内就能完全显示！
- 支持查询职业模板和可选额外技能包。
- 支持输出骰子属性设置指令

下载：前往 https://github.com/DeltaGreenCHN/Character-Sheet/releases/latest ，下载zip文件。
